## 注册绳网账号

这是你在绳网的唯一通行证，当然你也可以像法厄同一样注册新的账号~~doge~~，绳网使用Github Discussions作为数据库，所以你懂我意思吗，你必须得有一个**Github**账号。在你注册完Github账号之后，可参考以下步骤：

import { Steps, LinkCard, CardGrid } from '@astrojs/starlight/components';

<Steps>

1. 安装TamperMonkey浏览器扩展

   <CardGrid>

     <LinkCard title="Chromium内核扩展" href="https://www.crxsoso.com/webstore/detail/dhdgffkkebhmkfjojejmpbldmpobfkfo" />

     <LinkCard title="Firefox扩展" href="[Tampermonkey – 下载 🦊 Firefox 扩展（zh-CN） (mozilla.org)](https://addons.mozilla.org/zh-CN/firefox/addon/tampermonkey/)" />

   </CardGrid>

2. 安装TamperMonkey脚本：[绳网小助手](https://greasyfork.org/zh-CN/scripts/502874-%E7%BB%B3%E7%BD%91%E8%B7%A8%E5%9F%9F%E5%8A%A9%E6%89%8B)

   ![脚本安装1](https://vip.helloimg.com/i/2024/08/17/66c08719b7e1e.png)

   点击“安装此脚本”后不同浏览器的安装页面可能不一样，点击**安装**即可

3. 返回绳网后刷新，登录你已经**注册好**的Github账号并授权登录

   ![Github登录](https://vip.helloimg.com/i/2024/08/17/66c088688d292.png)

4. 注册完成，左上角显示的是当前登录的账号，绳网登记是账号**公开仓库**的数量

   ![用户信息卡片](https://vip.helloimg.com/i/2024/08/17/66c088eb5fe6b.png)

</Steps>

## 发帖

终于在绳网注册好自己的账号了，那么如何在绳网上发布~~灌水~~帖子呢

在页面的右下角，点击“写文章”按钮跳转到发帖页面，编辑与发布参照图2

![发帖](https://vip.helloimg.com/i/2024/08/17/66c08b4d10f84.png)

![发帖](https://vip.helloimg.com/i/2024/08/17/66c08be1bef78.png)

## 评论

在一个帖子中，内容与评论中间会有“写回复”按钮，点击跳转到发评论页面

![发评论](https://vip.helloimg.com/i/2024/08/17/66c08e1cb7055.png)

编辑方式与写文章基本相同，不用输入标题
