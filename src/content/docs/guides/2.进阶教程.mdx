终于可以在绳网上发自己的帖子了，但你不想学学高级的功能吗 doge

## 取消订阅

由于Github Discussions的设定，你发布了帖子或在一个帖子留言后，如果有绳匠在下面留言，你就会收到Github给你发的邮件，如果你不希望这样，可以**取消订阅**通知~~TD~~

![取消订阅](https://vip.helloimg.com/i/2024/08/17/66c0930dca3d1.png)

## 删除帖子

如果你是绳网管理员或帖子的**发布者**，可以点击“写回复”右侧的删除按钮进行删除

import { Aside } from '@astrojs/starlight/components';

<Aside type="caution" title="注意">

由于Github限制，只有仓库所有者和协作者可以删除帖子，所以绳网内提供的删除功能只是在绳网瀑布流中隐藏了帖子，并不会从根本上删除

</Aside>

## 举报帖子

如果你~~看哪个帖子不顺眼~~看到哪个帖子违反了[《》]()，可以点击“写评论”右侧的举报按钮进行举报，输入举报理由并二次确认后可以在站内提交

<Aside type="caution" title="注意">

当某个帖子被举报的次数达到了5次后，此帖就会在瀑布流中隐藏，需要不同Github账号举报，单人恶意举报无效

</Aside>

## 搜索

绳网还提供了搜索功能，可以筛选标题中的文本，点击右上角标签的“搜索”并输入搜索内容以使用

<Aside type="tip" title="提示">

搜索后瀑布流会仅剩标题中包含关键词的帖子，刷新后恢复



高级搜索筛选请到[Discussions](https://github.com/share121/inter-knot/discussions)

</Aside>
